ifconfig

MAC адрес лежит в /sys/class/net/[имя девайса]/adress

- /etc/sysconfig/network - файл содержит имя хоста, маршрут по умолчанию и дополнительные параметры сети.
- /etc/modprobe.conf - Файл содержит соответствие модулей ядра сетевым устройствам.
- /etc/hosts - Информация об IP-адресах и соответствующих им именах хостов
- /etc/resolv.conf - Указываются DNS-серверы, а также дополнительные параметры, связанные с настройкой DNS клиента
- /etc/rc.d/init.d/network - Инициализационный скрипт, использующийся для активации и деактивации сетевого стека ОС


### Описание параметров сети
NETWORKING - Следует ли активизировать сетевой стек (yes|no)
HOSTNAME - Имя хоста
GATEWAY - Маршрут по умолчанию

### Описание параметров сетевого интерфейса
BOOTPROTO - Определяет тип протокола автоматической настройки интерфейса(none|dhcp|bootp)
STARTMODE - Определяет будет ли активирован данный интерфейс в процессе загрузки ОС или нет. (onboot|manual)
IPADDR - IP адресс сетевого интерфейса
NETMASK - Маска подсети
NETWORK - Адресс подсети
BROADCAST - Широковещательный адрес подсети

## Основные утилиты
ping, traceroute, netstat, tcpdump

## ping
посылает ICMP-пакет ECHO_REQUEST и ждёт ECHO_RESPONSE.
В обработке её результата участвуют таблицы маршрутизации, физические компонены сетей и сетевые шлюзы.

Опция | Значение
---|--
-с | количество пакетов которые будут отправлены
-i | интервал в секундах
-s | размер посылаемого пакета в байтах


## Удалённый доступ
1. удалённое выполнение команд (ssh)
2. удалённое администрирование (ssh, vnc, rdesktop)
3. приём-передача данных (scp, lftp, rsync)

### ssh
/etc/ssh/ssh_config - файл настройки клиентской части OpenSSH
~/.ssh/known_hosts - список известных хостов. Пополняется после успешного ssh соединения с хостом.
